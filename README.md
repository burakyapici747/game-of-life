# game-of-life
JavaScript HTML Game of life 

## Config.js

**Game settings in `config.js`**

``` js
gameSettings.canvasSettings = {
    canvasWidth: 800,
    canvasHeight: 800,
    resolution: 8,
    refreshTime: 50,
};
gameSettings.canvasObjectSettings = {
    cellColor: "black",
};
```
