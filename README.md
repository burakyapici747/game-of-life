# game-of-life
JavaScript HTML Game of life 

## Config.js Color Map

You can change rectangle color and index from config.js.

**Color enum**

``` js
gameSettings.canvasSettings = {
    canvasWidth: 800,
    canvasHeight: 800,
    resolution: 8,
    refreshTime: 50,
};
gameSettings.canvasObjectSettings = {
    cellColor: "black",
};
```
